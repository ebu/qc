# EBU QC: Catalogue API

## About

Specification of an API for querying a catalogue of EBU QC Item Definitions (as implemented at [https://qc.ebu.io/](https://qc.ebu.io/))

## Main files

* **XML Schema** for catalogue API responses: [qc-catalogue-api-schema.xsd](qc-catalogue-api-schema.xsd)
* **Specification** of the catalogue API: [qc-catalogue-api.docx](qc-catalogue-api.docx)

ℹ️ A [**PDF of the latest qc-catalogue-api.docx**](https://ebu.github.io/qc/qc-catalogue-api/qc-catalogue-api.pdf) is available (generated from the head of the "main" branch)

## Approach to versioning of qc-catalogue-api-schema.xsd

* If qc-catalogue-api-schema.xsd is ever changed then the XML namespace will be changed
