This is a very basic tool for inspecting EBU QC Reports. It is not feature complete and should only be used for demonstration purposes. Data presented using this tool may not reflect the contents present in the input document. The validation results should not be taken as authoritative.
Designed for use in Chrome

This tool loads an EBU QC Report XML file and demonstrates the benefits of using this format, such as:

  • A custom view can be created for each workflow role that shows only the required details
  • QC Reports from all QC tools are presented uniformly
